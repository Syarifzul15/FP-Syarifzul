About

Halo kenalin nama saya Syarifzul Hidayat mahasiswa UPN Veteran Jakarta jurusan D3-Sistem Informasi semester 3. Saya biasa dipanggil Ijul atau Syarif, Saya lahir di Jakarta tanggal 15 Maret 2001, kebetulan waktu itu saya lahir agak beda,iya saya lahir sungsang.Saya menempuh pendidikan pertama saya di TK-Almuhajirin, lalu ketika menurut gurunya saya sudah cukup bandel, maka saya dilepas untuk menempuh pendidikan kedua saya di SD-Cipayung 1, dan selama 6 tahun guru saya sudah menyadari pubertas yang saya alami, karena itu saya disuruh menempuh pendidikan berikutnya di SMPN 3 Cibinong, karena saya tidak begitu betah jadi saya di smp hanya 3 tahun lalu saya memilih melanjutkan pendidikan saya ke SMKN 1 Cibinong, tidak ada alasan khusus kenapa saya memilih SMK, bukan karena SMK lebih bagus dari SMA, tapi karena mamah saya ingin saya bisa cepet kerja. Tapi pada akhirnya tetap saja dia menyuruh saya kuliah. Saya sempet gapyear satu tahun buat buka usaha, setelah itu saya memutuskan untuk kuliah. Alasan Saya milih jurusan sistem infrmasi karena ketika saya SMK saya memilih jurusan Teknik Komputer Jaringan, jadi saya pikir untuk melanjutkan sedikit ilmu yang saya punya dan agar tidak terlalu sulit memahami materi saat kuliah, tapi saya salah karena tetap saja materinya susah.

Jadi karena saya merasa sedikit kesusahan saya memutuskan mengikuti KSM Android. Dulu saya sangat bodoh di matkul web pemrograman karena saya sama sekali tidak mengerti ngoding, tapi setelah saya menemukan KSM Android dan Bang Jovan Alhamdulillah ilmu saya bertambah dan menjadi percaya diri bahwa saya bisa magang di gugel, Enggak si sebenernya saya cuma bercanda saya belum terlalu ngerti banyak soal ngoding, ini aja mau final project ksm saya paniknya dari sebelum masuk ksm. jadi semoga nanti pas final project saya bisa menjawab semua pertanyaan yang diberikan, doain ya mah semoga ijul bisa menghadapi ini huhu. Oiya saya yakin si sebenernya ini gaakan dibaca juga, tapi kalo pun dibaca, saya mau nitip salam buat mas Hanif, Makasih mas udah ngenalin KSM android, makasih gapernah marah kalo ditanya tanya, intinya makasih selalu ada mas:(.
